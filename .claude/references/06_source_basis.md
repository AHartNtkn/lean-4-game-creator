## Educational sources used conceptually
- work on worked examples and example-based learning
- interleaved mathematics practice
- retrieval practice
- mastery / formative feedback
- papers on Lean and proof education
- Patrick Massot’s work on Verbose Lean and paper-proof transfer

The pack is therefore grounded in:
1. the platform’s actual mechanics,
2. actual games on that platform,
3. actual course structure from a real class,
4. and general proof/pedagogy literature.

It is intended to be judgment-heavy rather than doc-heavy.
