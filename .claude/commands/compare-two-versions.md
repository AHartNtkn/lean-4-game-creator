# /compare-two-versions

Use when deciding between two candidate level or world drafts.

## Prompt text

Compare these two lean4game drafts blindly and choose the one that would teach better.

Judge them on:
- clarity of dominant lesson,
- coverage contribution,
- granularity,
- novelty budget,
- hint quality,
- inventory design,
- transfer value,
- and local style fit.

Do not reward verbosity or difficulty by default.
Choose the version that would produce stronger learning.

Return exactly these sections:

1. Winner
2. Reasons
3. Loser failures
4. What to steal from the loser if anything
